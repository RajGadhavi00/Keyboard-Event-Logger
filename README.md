# Keyboard Event Logger

## Overview

This project is a simple Keyboard Event Logger built using Python and Tkinter. It records the keys typed inside the application's text area and displays them with timestamps. Users can start or stop logging, clear the recorded data, and save the log to a text file.

## Features

- User-friendly GUI using Tkinter
- Start and Stop Logging
- Records keyboard events inside the application
- Displays timestamp for every key press
- Save logs to a text file
- Clear recorded logs
- Live key counter

## How to Use

1. Launch the application.
2. Click **Start Logging**.
3. Type inside the application's text area.
4. The recorded keys will appear with timestamps.
5. Click **Save Log** to export the log as a text file.
6. Use **Stop Logging** to pause recording.
7. Click **Clear Log** to remove all recorded entries.
